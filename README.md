# Supplementary Materials

This repository contains supplementary materials associated with a manuscript
submitted to *Agriculture* (MDPI).

## Author
Doo-Ho Choi

## Contents
- scripts/: Python scripts used for YOLO-based mushroom phenotypic analysis
- data/: Processed datasets used in the study
- results/: Supplementary figures and tables
- supplementary/: Supplementary materials referenced in the manuscript

## Data Availability
Raw image data are not publicly available due to storage limitations and ongoing research.
Processed data and scripts sufficient to reproduce the analysis are provided.
